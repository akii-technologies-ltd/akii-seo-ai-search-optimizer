## Body

We just open-sourced something I wish existed two years ago.

For the past 18 months, I've watched SaaS founders rank #1 on Google for their core keyword — and still get crushed when buyers ask ChatGPT or Claude or Gemini for "the best tool for X."

Their brand is never named. A competitor is recommended instead.

It's because the AI engines don't read Google. ChatGPT reads Bing-sourced "best of" list articles and review platforms. Gemini reads Google but hard-excludes anyone below 3.5★ aggregate reviews. Perplexity weights top-5 Google list mentions at 64%. Claude leans 68% on trusted business databases like Hoovers, Bloomberg, and IBISWorld.

Each engine. Different algorithm. Different fix path.

So we built a free Claude Code plugin that encodes all of it.

**Akii — SEO & AI Search Optimizer**

→ 13 skills that auto-trigger from natural language
→ 5 autonomous agents for end-to-end audits and content strategy
→ 8 slash commands for focused tasks
→ The Princeton/IIT-Delhi/Allen Institute GEO method (validated +40% AI visibility lift) built in
→ Auto-detects your existing Ahrefs, Google Search Console, DataForSEO, and Apify MCPs for real-data outputs
→ A `/ai-visibility-score <domain>` command that returns the same 0–100 score akii.com customers track 24/7
→ MIT-licensed. No login. No signup. No usage cap.

Five concrete things this unlocks for your team:

1. **Pre-launch audits inside your IDE.** Run `/check-file` before deploying a landing page. Catches missing schema, weak meta, AEO chunk-quality issues.

2. **Per-engine visibility maps.** Ask Claude "how does my brand rank across ChatGPT, Gemini, Perplexity, and Claude?" — get a 30-day fix plan tailored to each engine.

3. **Princeton-method GEO rewrites.** Take any blog post; apply the empirically validated tactics (citations / expert quotes / statistics / fluency optimization / authoritative tone) automatically.

4. **llms.txt generation.** The emerging standard for telling AI crawlers what content matters. Generated from your sitemap in one command.

5. **Bulk schema across your site.** The `schema-generator` agent works through entire blog directories with one approval, AEO-optimized fields included (sameAs, granular LocalBusiness subtypes, actionableFeedbackPolicy).

Install (you'll need Claude Code from claude.ai/code):

```
/plugin marketplace add akii-technologies-ltd/akii-seo-ai-search-optimizer
/plugin install akii-seo-ai-search-optimizer@akii
```

Then ask Claude "audit my site" or run `/ai-visibility-score yourcompany.com`.
